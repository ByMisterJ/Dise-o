Muestra/Oculta el texto
Una Página Web es un documento electrónico que forma parte de la WWW (World Wide Web) generalmente construido en el lenguaje HTML (Hyper Text Markup Language o Lenguaje de Marcado de Hipertexto) o en XHTML (eXtensible Hyper Text Markup Language o Lenguaje de Marcado de Hipertexto Extensible).

Tecnologías

HTML
El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos.
CSS
CSS son las siglas en inglés para hojas de estilo en cascada (Cascading Style Sheets). Básicamente, es un lenguaje que maneja el diseño y presentación de las páginas web, es decir, cómo lucen cuando un usuario las visita. Funciona junto con el lenguaje HTML que se encarga del contenido básico de los sitios. Se les denomina hojas de estilo en cascada porque puedes tener varias y una de ellas con las propiedades heredadas (o «en cascada») de otras. Para muchas personas, una simple plantilla de blog es suficiente. Aun así, cuando quieras personalizar la apariencia de un sitio, necesitarás implementar CSS que, en conjunto con un buen CMS, te ayudará a potenciar el alcance de tu contenido.
JavaScript
JavaScript es un lenguaje de programación o de secuencias de comandos que te permite implementar funciones complejas en páginas web, cada vez que una página web hace algo más que sentarse all y mostrar información estática para que la vias, muestra oportunas actualizaciones de contenido, mapas interactivos, animación de Graficos 20/30, desplazamiento de máquinas reproductoras de video, etc., puedes apostar que probabilemente JavaScript está involucrado. Es la tercera capa del pastel de las tecnologies web estándar, dos de las cuales (HTML y C55) hemes cubierto con mucho más detalle en otras partes del Área de aprendizaje.
